## 1. 项目结构速览

```
vision-hub/
├── backend/              # 后端源码（唯一源头）
├── deploy/               # 云端部署版（rsync 直推服务器）
├── submission/           # 提交评委版（三端完整+APK）
├── app/                  # Android 本地测试版
├── hardware/             # 硬件固件（ESP32 + K210）
├── docs/                 # 项目文档
├── server-deploy.sh      # ★ 一键部署脚本
└── web-register.json     # 服务器部署注册
```

**三端通信架构**：

```
硬件(ESP32+K210) ──WiFi──→ 后端(Go) ←──HTTP── Android APP
       ↑ 离线本地推理            ↑ 云端中转          ↑ 监护端
```

**三版本架构**：

| 版本 | 文件夹 | 用途 |
|------|--------|------|
| 本地测试版 | `backend/` + `app/` | 日常开发（127.0.0.1） |
| 云端部署版 | `deploy/` | rsync 推送服务器（47.94.146.53） |
| 提交评委版 | `submission/` | 三端完整源码+APK（交给评委） |

## 3. 一键部署（推荐）

改完 `backend/` 代码后，在项目根目录执行：

```bash
./server-deploy.sh
```

脚本自动完成三步：

```
① backend/ ──rsync──→ deploy/（本地同步）
② deploy/   ──rsync──→ 服务器 /opt/visionguard/deploy/
③ SSH 服务器 → docker compose -f docker-compose.prod.yml up -d --build
```

## 4. 服务器首次初始化

如果服务器还没有部署目录，需要先初始化一次：

```bash
# SSH 登录
ssh root@47.94.146.53

# 创建部署目录
mkdir -p /opt/visionguard/deploy

# 从旧容器提取 .env（如果之前部署过）
docker inspect cloud-deploy-backend-1 \
  --format '{{range .Config.Env}}{{println .}}{{end}}' \
  | grep -vE '^(PATH=|TZ=)' > /opt/visionguard/deploy/.env

# 首次手动推送
# （在本地执行）
rsync -avz --delete \
  --exclude='.env' --exclude='.env.example' \
  deploy/ root@47.94.146.53:/opt/visionguard/deploy/

# 启动
ssh root@47.94.146.53 "cd /opt/visionguard/deploy && docker compose -f docker-compose.prod.yml up -d --build"
```

`.env` 关键配置（Docker 内部用服务名）：

```ini
SERVER_PORT=3000
DB_HOST=postgres          # ← Docker 服务名，不是 localhost
DB_PORT=5432
DB_USER=visionhub
DB_PASSWORD=<你的密码>
DB_NAME=visionhub
REDIS_HOST=redis           # ← 同上
REDIS_PORT=6379
JWT_SECRET=<你的随机密钥>
```

## 8. 常见问题

### Q: 后端启动后连不上？
```bash
docker compose -f docker-compose.prod.yml ps
docker compose -f docker-compose.prod.yml logs backend
```

### Q: 数据库连接失败？
检查 `.env` 中 `DB_HOST=postgres`（不是 `localhost`）。

### Q: ESP32 连不上？
1. 确认 WiFi 已连接（2.4GHz，WPA2，英文 SSID）
2. `ping 47.94.146.53`

### Q: 想回到本地开发？
- 后端 `.env`：`DB_HOST=localhost`，`REDIS_HOST=localhost`
- Android：`BASE_URL = "http://127.0.0.1:3000/"` + `adb reverse tcp:3000 tcp:3000`
