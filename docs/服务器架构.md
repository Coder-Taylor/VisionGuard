## 目录层

```
/srv/web/                      ← Web 根目录（规划）
├── public/                    公开
└── internal/                  密码保护

/opt/visionguard/              ← VisionGuard 项目
├── repo/                      Git 仓库
└── deploy/                    部署运行目录
```

## VisionGuard 部署

```bash
# 本地 → 服务器
./server-deploy.sh

# 服务器上
cd /opt/visionguard/deploy && bash deploy.sh
```

## 开发→部署流程

```
本地开发 → server-deploy.sh → git push Gitee → 服务器 git pull → Docker 重建
```
