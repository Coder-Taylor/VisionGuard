## Nginx 路由表

| 路径 | 目标 | 说明 |
|------|------|------|
| `/` | 静态文件 `/srv/web/public/` | 个人站首页 |
| `/apps/*` | 静态文件 `/srv/web/public/apps/` | 网页合集 |
| `/api/*` | → `127.0.0.1:8080` | 个人站 API |
| `/vg/api/*` | → `127.0.0.1:3000`（反向代理） | VisionGuard API |
| `/vg/app/*` | 静态文件（Web dist/） | VisionGuard Web 管理后台（待部署） |
| `/vg/` | 静态文件（旧管理面板） | VisionGuard 旧管理页 |

## VisionGuard 部署

```bash
# 后端 → 服务器
./server-deploy.sh

# Web → 服务器
cd submission/web && npm run build
scp -r dist/* root@47.94.146.53:/srv/web/public/vg/app/
```

## 开发→部署流程

```
本地开发 → server-deploy.sh → git push Gitee → 服务器 git pull → Docker 重建
Web 开发 → npm run build → scp dist/ → 服务器 Nginx 静态文件
```
