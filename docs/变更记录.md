## [v1.4.4] — 2026-05-06 深夜

### Fixed
- **OCR 硬件轮询 GET 401**（严重 bug）：Fiber 路由 `/api/v1/ocr/result/:taskId`（参数路由, userAuth）注册在 `/api/v1/ocr/result/latest`（精确路由, deviceAuth）之前，导致 `latest` 被 `:taskId` 匹配到 userAuth 中间件，设备 JWT 通不过用户认证 → Android 永远显示"等待中"。修复：精确路由移到参数路由前。
- **drugName UTF-8 截断乱码**：豆包返回中文 drugName 时 byte-based 截断切到多字节字符中间 → DB UTF-8 编码错误。改为 rune-based 截断。
