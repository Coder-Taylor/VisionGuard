## 服务架构

```
                     ┌──────────────┐
                     │   Nginx :80   │
                     └──┬───┬───┬───┘
        ┌───────────────┤   │   ├───────────────┐
        ▼               ▼   │   ▼               ▼
   静态文件          API :8080  │   WS :8080     内部模块
 /srv/web/public/    (Go)  │   (Go)       /internal/ (密码)
        │                    │                    │
   ┌────┴────┐        web-backend          ┌─────┴─────┐
   │         │       (systemd)             │           │
  apps/   assets/                     admin/    visionguard/
   │         │                         (空)     (空，占位)
   ├─about/  └─images/
   ├─tools/answer/             Docker :3000        Docker :5432
   ├─login/               cloud-deploy-backend-1   postgres:16-alpine
   ├─blog/                  (Vision-Hub Go)        (共享数据库)
   ├─games/                       
   ├─projects/              Docker :6379
   └─resources/            redis:7-alpine
```

## 端口分配

| 端口 | 进程 | 用途 | 外部可达 |
|------|------|------|----------|
| 80 | nginx | 入口代理 | ✅ |
| 3000 | docker-proxy (Vision-Hub) | 仅内部 Nginx 代理 | ❌ UFW 应阻止 |
| 5432 | docker-proxy (PostgreSQL) | 共享数据库 | ❌ |
| 6379 | docker-proxy (Redis) | 共享缓存 | ❌ |
| 8080 | web-backend (Go) | 主站 API + WebSocket | ❌ 127.0.0.1 仅 |

## Nginx 路由表

| 路径 | 目标 | 限流 | 说明 |
|------|------|------|------|
| `/` | 静态文件 `/srv/web/public/` | — | SPA fallback |
| `/apps/*` | 静态文件 `/srv/web/public/apps/` | — | 网页合集 |
| `/assets/*` | 静态文件 `/srv/web/public/assets/` | 30d 缓存 | 图片等资源 |
| `/api/login` | → `127.0.0.1:8080` | 3r/m | 登录 |
| `/api/register` | → `127.0.0.1:8080` | 3r/m | 注册 |
| `/api/*` | → `127.0.0.1:8080` | 10r/s | 通用 API |
| `/ws/*` | → `127.0.0.1:8080` | 5 连接 | WebSocket |
| `/internal/*` | `/srv/web/internal/` | — | 密码保护 |
| `/vg/api/*` | → `127.0.0.1:3000` | — | VisionGuard API（反向代理） |
| `/vg/app/*` | 静态文件（Web dist/） | — | VisionGuard Web 管理后台（待部署） |

## 访问权限矩阵

| 模块 | 访问级别 | 认证方式 |
|------|----------|----------|
| 首页 `/` | 公开 | 无 |
| 知答刷题 `/apps/tools/answer/` | 公开 | 无 |
| 关于我 `/apps/about/` | 公开 | 无 |
| 登录 `/apps/login/` | 公开 | JWT 登录 |
| 博客/游戏/项目/资源 | 公开（待建） | 无 |
| API `/api/*` | 部分公开 | JWT（部分接口） |
| WebSocket `/ws/quiz` | 公开 | WebSocket 连接 |
| 内部模块 `/internal/*` | 密码保护 | HTTP Basic Auth |
| Vision-Hub `/internal/vg/api/*` | 密码保护 | HTTP Basic Auth |
| VisionGuard Web `/vg/app/*` | 公开（待部署） | 无 |

## 服务管理命令

```bash
# Web 后端
systemctl status web-backend
systemctl restart web-backend
journalctl -u web-backend -f

# 重新部署
cd /home/admin/web && bash deploy/deploy.sh

# Vision-Hub（独立管理）
cd /home/admin/web/visionguard && bash server-deploy.sh
```
